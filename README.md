# BlinkShortener
This is my linkshortener Web App, supporting Base62 encoding, efficient database indexing and fast redirects. Designed to handle massive scale (billions of URLs), with clean API endpoints and easy deployment.

Current Features:
1. Generate short URLs using Base62 encoding
2. Fast redirects with indexed lookups
3. Custom aliases
